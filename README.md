# koa-server-demo

### 简介
  + 利用 koa2 简单搭建的一个提供数据接口的服务器 Demo.
  + 数据库采用的 MySQL.

### 安装和使用
```
  // 下载
  git clone 'git@github.com:NathanWorld/koa-server-demo.git'
  // 切换到项目目录
  cd koa-server-demo
  // 安装依赖包
  yarn 
  // 运行
  yarn start
  // 访问链接, 在Chrome浏览器中输入如下地址
  http://localhost:3000
```
